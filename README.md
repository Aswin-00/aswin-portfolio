## About The Project

This is my personal portfolio website designed and planned entirely by me.

The website follows a monitoring-dashboard inspired interface to reflect my background in Cloud Engineering, DevOps, infrastructure monitoring, automation, and incident handling.

Instead of a traditional portfolio layout, the platform is built to feel like a live operational system with telemetry-style navigation, system states, and terminal-inspired interactions.
